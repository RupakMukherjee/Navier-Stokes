Navier-Stokes
===============================================

Multi dimensional Navier-Stokes Equation solver using Pseudo-spectral algorithm.

Installation
------------
#### Common Prerequisites
1. gfortran compiler for fortran95 (for serial codes)
2. FFTW library (fftw3)
3. git
